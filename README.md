# HELP-DESK-MANAGEMENT-SYSTEM

## About The Project
A Java-based Help Desk Ticket Management System that streamlines the process of raising, tracking, and resolving support requests. The system provides role-based access for Users, Support Staff, and Administrators, ensuring efficient ticket handling and management.

## Features

* Secure Login & Registration
* Role-Based Access Control
* Create and Manage Support Tickets
* Update Ticket Status (Open → In Progress → Closed)
* Admin Dashboard for Monitoring Activities
* Audit Logs and Activity Tracking
* Basic Analytics and Reporting
* File-Based Data Persistence

## Tech Stack

* Java
* Object-Oriented Programming (OOP)
* File Handling
* Collections Framework
* Swing/Java GUI

## Architecture

The project follows a layered architecture:

```
UI Layer
   ↓
Service Layer
   ↓
Repository Layer
   ↓
File Storage
```

This separation improves maintainability, scalability, and code organization.
